# Additive model Multiplicative model
---

T S Iの成分をすべて足したのが加法モデル
$Y=T+S+IY = T + S + I$

T S Iの成分をすべて掛けたたのが加法モデル
$Y=T×S×IY = T \times S \times I$

T S Iの成分を足し算したり掛け算したのが混合モデル
$Y=T+S×IY = T \times S \times I$
